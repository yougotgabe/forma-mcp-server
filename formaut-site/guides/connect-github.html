Setup guide — Step 1 of 3

Connect your GitHub account

GitHub is where your site's code lives. Every change Formaut makes gets saved here automatically — so your site is always backed up and you can see exactly what was changed and when. You own the repo. If you ever leave Formaut, the code comes with you.

1
Create a GitHub account
Go to github.com and sign up with your business email. If you already have an account, skip to step 2.

Use your business email — not a personal one. GitHub will ask you to verify the email before you can do much else.
Already have GitHub? Great — skip straight to step 3. You don't need a new account.
2
Verify your email
GitHub sends a verification email immediately after signup. Open it and click the confirmation link. You won't be able to create repos or authorize apps until this is done.

Check spam if it doesn't arrive in a minute or two.
3
Generate a personal access token
Formaut needs a token to create and manage your site repo on your behalf. Here's exactly where to find it:

GitHub → your avatar (top right) → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic)
What to fill in
Note: Formaut — site access
Expiration: No expiration
Scopes to check: repo (the whole top-level checkbox — this includes everything under it)
Important: Copy the token immediately after generating it. GitHub only shows it once. If you close the page before copying, you'll need to generate a new one.
4
Paste the token into Formaut
Go back to your setup page and paste the token into the GitHub token field. Formaut encrypts it immediately — it's never stored in plain text, and not even Formaut's team can read it once it's saved.
Why GitHub?

Every website is code. That code needs to live somewhere. GitHub is the industry standard for storing and versioning code — it's used by everyone from solo developers to major companies.

More importantly for you: it means your site isn't trapped inside Formaut's systems. The code is in your account. If you stop using Formaut, hire your own developer, or just want to move things somewhere else — everything you need is right there.

The repo Formaut creates is private by default. Only you and Formaut can see it.
