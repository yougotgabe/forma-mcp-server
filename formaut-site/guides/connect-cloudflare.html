Setup guide — Step 2 of 3

Connect your Cloudflare account

Cloudflare is where your site actually runs — it hosts your pages, handles your domain, and runs the server functions that power your admin panel. It's also one of the few services that has no bandwidth limits on the free tier, which means your site won't slow down or get throttled if you have a busy day.

1
Create a Cloudflare account
Go to cloudflare.com and sign up. If you already have an account, skip to step 2.

The free plan is all you need — Formaut works entirely within Cloudflare's free tier limits.
Already have Cloudflare? Skip straight to step 3.
2
Find your Account ID
You'll need your Account ID — it's a long string of letters and numbers that identifies your Cloudflare account.

Cloudflare dashboard → Workers & Pages (left sidebar) → Overview → Account ID (right sidebar)

Copy it — you'll paste it into Formaut in a moment.
3
Create an API token
The API token is what lets Formaut deploy your site and make changes on your behalf.

Cloudflare dashboard → My Profile (top right avatar) → API Tokens → Create Token
Exact settings to use
Start from template: Edit Cloudflare Workers
Then add one permission: Cloudflare Pages → Edit
Account Resources: Include → All accounts
Zone Resources: Include → All zones
TTL / Expiration: Leave blank (no expiry)
Important: Copy the token immediately. Cloudflare only shows it once. If you miss it, delete the token and create a new one — the old one can't be recovered.
4
Paste both values into Formaut
Head back to your setup page. You'll paste:

1. Your Account ID into the Account ID field
2. Your API token into the API token field

Both are encrypted on arrival. Neither is stored in plain text.
Why Cloudflare?

Cloudflare runs on a global network with over 300 data centers. That means your site loads fast for visitors anywhere in the world — not just near a single server.

We looked at Netlify and Vercel. Both have bandwidth caps on free tiers that become a problem once a site gets traffic. Cloudflare Pages has no bandwidth limit. For a local business that has a good month, that matters.

Cloudflare also handles your domain's DNS, your SSL certificate, and DDoS protection automatically. It's a lot of infrastructure that would otherwise cost hundreds of dollars a month, included for free.
